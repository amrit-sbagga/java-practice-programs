# Practice Programs (interview coding)

Empty Java templates with **requirements only** — you implement the logic.

Files use the **default package** (no `package` line) so names stay simple: `Fibonacci.java`, `IsPrime.java`, etc.

## Run from terminal

```bash
cd java-practice-programs
mvn -q compile exec:java -Dexec.mainClass=Fibonacci
```

Or run `main` from your IDE after opening this folder as part of the Maven project.
